# basic-vue-ts

Real Estate List Application - A Vue.js 3 project using Primevue theme

## Libraries
motion
tailwind


## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
